Hệ điều hành: Window

Phần mềm cần cài đật:
- Visual Studio (có .NET Framework, IIS Express)
- SQL Server
- MSSQL Server Management Studio

Tạo database:
- Mở file ./QUAN_LY_HOC_SINH/GenerateData.sql bằng MSSQL Management Studio
- Chạy file để tạo database

Build và chạy ứng dụng:
- Mở file ./QUAN_LY_HOC_SINH/QUAN_LY_HOC_SINH.sln
- Build và chạy ứng dụng
